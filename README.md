# Online Fashion Store(E-commerce Website)
Made with ReactJS,Redux Strapi(with MySQL Database)

## Features :
- Add to Cart
- Featured Products
- Payment Integration Using Stripe API
- Product Categories and Filters for each Category
- User Authentication

## Start React App
`cd client`  
`npm start`

## Start Strapi Server
`cd api`  
`npm run develop`
